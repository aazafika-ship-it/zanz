1. Apa itu ADB dan Fastboot?

ADB (Android Debug Bridge) dan Fastboot adalah dua *command-line tool* yang esensial bagi siapa pun yang ingin memodifikasi atau mengelola perangkat Android mereka secara mendalam. Keduanya merupakan bagian dari paket Platform-Tools yang dirilis resmi oleh Google.

  • ADB: Digunakan untuk berkomunikasi dengan perangkat Android yang berjalan normal (atau dalam mode *recovery*).
  • Fastboot: Digunakan untuk memodifikasi *firmware* perangkat, seperti *flashing* *custom recovery*, *unlock bootloader*, atau *flashing factory image*.

Kenapa Ini Wajib Diinstal? Tanpa kedua *tool* ini, Anda tidak akan bisa melakukan modifikasi tingkat lanjut seperti *Rooting* atau *Custom ROM Installation*.

‚ö†Ô∏è **DISCLAIMER PENTING**

Melakukan modifikasi perangkat Android menggunakan ADB dan Fastboot memiliki risiko, termasuk **kehilangan data** dan **brick** jika langkah yang dilakukan salah.

Segala risiko dan konsekuensi yang timbul dari panduan ini sepenuhnya menjadi tanggung jawab Anda. Pastikan Anda telah melakukan **backup data** sebelum memulai.

3. Checklist Persiapan Sebelum Instalasi

  1. ‚úÖ **PC/Laptop Windows:** Pastikan Anda memiliki akses Administrator.
  2. ‚úÖ **Driver Perangkat:** Pastikan driver USB HP Anda sudah terinstal.
  3. ‚úÖ **Mode Developer Option:** Aktifkan **USB Debugging** dan **OEM Unlocking** di Pengaturan HP Anda.

4. Langkah-Langkah Instalasi Platform-Tools

4.1. Download Platform-Tools Resmi

Unduh file Platform-Tools terbaru langsung dari sumber resmi Google (Android Developers):

üì• DOWNLOAD PLATFORM-TOOLS

4.2. Ekstrak File ke Lokasi Mudah

Ekstrak file ZIP ke lokasi yang mudah diakses, misal:

C:\adb\

Pastikan folder platform-tools yang berisi file adb.exe berada langsung di dalam C:\adb.

4.3. Menambahkan ke Variabel Lingkungan (Path) Windows (KRUSIAL)

Agar Anda bisa menjalankan ADB/Fastboot dari mana saja:

  1. Buka **Environment Variables** (Variabel Lingkungan) Windows.
  2. Di bagian **System variables**, cari dan klik dua kali pada variabel bernama **Path**.
  3. Klik tombol **"New"** dan masukkan path folder Anda:
    4. C:\adb\platform-tools
  4. Klik **OK** di semua jendela.

5. Cara Tes Instalasi ADB & Fastboot

Buka CMD baru. Ketik perintah berikut:

adb --version
adb devices

Jika berhasil, output akan menampilkan versi dan **serial number** perangkat Anda diikuti **device**.

Proses Unlock Bootloader akan menghapus (Wipe) semua data di perangkat Anda dan dapat membatalkan garansi. Pastikan Anda sudah mem-backup seluruh data penting.

Kami tidak bertanggung jawab atas *brick* atau kerusakan yang mungkin terjadi pada perangkat Anda.

Langkah 1: Cek Koneksi ADB

Buka Command Prompt (CMD) atau PowerShell. Jalankan perintah ini untuk memverifikasi perangkat terdeteksi:

adb devices

Pastikan output menampilkan status **device**.

Langkah 2: Masuk ke Fastboot Mode

Reboot perangkat Anda ke mode Fastboot (Bootloader Mode) menggunakan perintah:

adb reboot bootloader

Verifikasi di Fastboot Mode dengan:

fastboot devices

Langkah 3: Jalankan Perintah Unlock Bootloader

Jalankan perintah inti untuk *unlock*. **Proses ini menghapus data!**

fastboot flashing unlock

Pilih **"YES"** pada konfirmasi yang muncul di layar HP Anda.

Langkah 4: Reboot dan Selesai

Setelah proses selesai, *reboot* perangkat Anda:

fastboot reboot
